# Algoritmos e Estruturas de Dados Avançadas

Este repositório contém implementações de estruturas de dados complexas e algoritmos desenvolvidos na linguagem C/C++ para a disciplina de Algoritmos e Estruturas de Dados na **EACH-USP**. O foco destas implementações é a manipulação direta de memória, uso intensivo de ponteiros e otimização de performance.

## 🧠 Implementações de Destaque

### 1. Teoria dos Grafos & Busca em Largura (BFS)
Solução para o problema de encontrar vértices equidistantes a dois pontos de origem em um grafo não direcionado.
* **Técnicas aplicadas:** Modelagem através de Listas de Adjacência para economia de memória em grafos esparsos e implementação de Filas (Queues) dinâmicas do zero para controle estrutural da travessia em nível (BFS).

### 2. Árvore Trie (Árvore de Prefixos)
Implementação de uma estrutura de dados de recuperação de informações (Trie) focada em dicionários e busca de palavras.
* **Técnicas aplicadas:** Alocação dinâmica agressiva (`malloc`/`free`), recursão profunda e manipulação de ponteiros duplos (`NO** filhos`) para inserção, busca e remoção otimizada de strings.

### 3. Listas Duplamente Ligadas
Sistema de gerenciamento e classificação de times baseado em regras de desempate complexas.
* **Técnicas aplicadas:** Construção do zero do sistema de nós, controle rigoroso de ponteiros direcionais (`ant` e `prox`) e lógica de ordenação e inserção in-place.

## 🚀 Tecnologias
* **Linguagens:** C / C++
* **Conceitos Chave:** Gerenciamento Manual de Memória, Alocação Dinâmica, Estruturas de Dados Não-Lineares.
